# Blog Design Research: Ultrarunning/Sports Personal Blog

Research compiled March 2026. Focused on award-winning, minimal, dark-themed, and sports-adjacent blog designs.

---

## 1. Award-Winning & Critically Acclaimed Personal Sites

### Tobias van Schneider — vanschneider.com
- **URL**: https://vanschneider.com/
- **Why notable**: Awwwards-recognized personal site from the designer behind Semplice, Carbonmade, and mymind. Frequently cited as a gold standard for personal blog design.
- **Design elements worth borrowing**:
  - Dark background with high-contrast white typography (Maison Neue font family)
  - Blog posts surfaced directly on the homepage — no extra click to discover content
  - Fullscreen live search that doubles as a content discovery tool
  - Moody, atmospheric imagery with generous spacing
  - Minimal navigation; the content IS the interface

### Rauno Freiberg — rauno.me
- **URL**: https://rauno.me/
- **Why notable**: Interaction designer at Vercel. Site won recognition on Killer Portfolio. The entire site is built around an OS metaphor with a dock, ambient sounds, and fluid transitions.
- **Design elements worth borrowing**:
  - Dark mode with carefully chosen tonal values that complement photography
  - Side-scrolling project/photo gallery on the homepage
  - Playful dock animation inspired by macOS
  - Photography page: full-screen grid with perfectly sized images
  - Subtle interaction sounds that add personality without annoyance
  - The "Craft" section (rauno.me/craft) is an exemplary format for longer essays with inline demos

### Paco Coursey — paco.me
- **URL**: https://paco.me/
- **Why notable**: Webmaster at Linear, previously built Vercel's design system. His personal site is a masterclass in restrained dark-mode design.
- **Design elements worth borrowing**:
  - Extremely minimal dark UI — almost no visual chrome
  - System-font-forward typography (or very clean sans-serif)
  - Content-first layout: just text, dates, and links
  - Subtle hover states and transitions
  - Proves that a blog can be compelling with almost zero decoration

### Lee Robinson — leerob.com
- **URL**: https://leerob.com/
- **Why notable**: VP of Product at Vercel (formerly). His site is a widely-referenced example of a developer/writer personal blog. Built with Next.js + MDX + Tailwind.
- **Design elements worth borrowing**:
  - Clean, near-monochrome palette with dark mode support
  - View Transitions API for smooth page-to-page animations
  - Blog post list is simple: title + date, no thumbnails or cards
  - Minimal header, no sidebar, no clutter
  - "Stack" page showing tools used — a nice personal touch

## 2. Content-Focused Blogs with Excellent Design

### Farnam Street — fs.blog
- **URL**: https://fs.blog/
- **Why notable**: One of the most respected long-form blogs online (100k+ subscribers). Design serves the writing without competing with it.
- **Design elements worth borrowing**:
  - Generous whitespace and calm color palette
  - Clean serif fonts optimized for long-form readability
  - Structured formatting (pull quotes, section headers, highlighted boxes)
  - No visual clutter — no ads, no pop-ups, no sidebar widgets
  - Light theme but the restraint and focus is transferable to dark

### Notion Blog — notion.so/blog
- **URL**: https://www.notion.so/blog
- **Why notable**: Black-and-white minimalism that matches Notion's brand. Frequently praised for its clean editorial layout.
- **Design elements worth borrowing**:
  - Monochrome palette (black, white, subtle grays)
  - Card-based layout with consistent sizing
  - Typography-driven hierarchy — size and weight do all the work
  - No decorative elements; imagery is purposeful

### Zen Habits — zenhabits.net
- **URL**: https://zenhabits.net/
- **Why notable**: Recognized by Time Magazine as a Top 25 Blog. The design IS the message: radical minimalism for a blog about simplicity.
- **Design elements worth borrowing**:
  - Almost no design at all — just text on a page
  - Proof that extreme restraint can be a bold design choice
  - No images, no sidebar, no categories — just the post
  - Good reference point for "how minimal can you go?"

## 3. Sports, Running & Outdoor Niche

### iRunFar — irunfar.com
- **URL**: https://www.irunfar.com/
- **Why notable**: The definitive ultrarunning and trail running publication. The most-read site in the niche. Worth studying as the benchmark audience expectation.
- **Design elements worth borrowing**:
  - Strong editorial hierarchy: featured articles, category sections, sidebar
  - Heavy use of race/trail photography (full-width hero images)
  - Review format with structured sections and comparison tables
  - Content density done right — lots of content but well-organized
  - Note: design is functional rather than "award-winning" — there is room to differentiate

### Trail Runner Magazine — trailrunnermag.com
- **URL**: https://www.trailrunnermag.com/
- **Why notable**: Major print+digital trail running publication. Magazine-style web layout.
- **Design elements worth borrowing**:
  - Magazine grid layout with varying card sizes for visual rhythm
  - Strong photography integration — every article has a compelling hero image
  - Category color-coding
  - Good example of balancing content volume with visual appeal

### Kilian Jornet — kilianjornet.cat
- **URL**: https://www.kilianjornet.cat/
- **Why notable**: Personal site of arguably the greatest mountain athlete ever. Bilingual (Catalan/English). Separate news subdomain (news.kilianjornet.cat).
- **Design elements worth borrowing**:
  - Full-bleed mountain/trail photography as hero sections
  - Clean separation between personal brand, news/blog, and equipment sections
  - Multilingual support done cleanly
  - Shows how an athlete's personal site can be both brand and blog

### Courtney Dauwalter — courtneydauwalter.com
- **URL**: https://www.courtneydauwalter.com/
- **Why notable**: Personal site of one of ultrarunning's biggest stars. Simple brand-forward athlete site.
- **Design elements worth borrowing**:
  - Personality-forward design — the site feels like her (fun, approachable)
  - Clear calls-to-action (race schedule, partnerships)
  - Shows the athlete-blog archetype in the ultrarunning space

## 4. Dark-Mode Design References (Non-Blog)

These are not blogs but are excellent references for dark-mode aesthetics to apply to a blog.

### Linear — linear.app
- **URL**: https://linear.app/
- **Why notable**: Widely considered the gold standard of dark-mode SaaS UI. Designed by Paco Coursey and team.
- **Design elements worth borrowing**:
  - Subtle dark shades with soft gradients (not pure black)
  - Muted accent colors that don't overwhelm
  - Micro-animations and transitions that feel polished
  - Typography that remains highly readable on dark backgrounds

### Vercel — vercel.com
- **URL**: https://vercel.com/
- **Why notable**: Performance-first dark UI. Their design system (vercel.com/design) is publicly documented.
- **Design elements worth borrowing**:
  - Near-black backgrounds (#000 or very close) with white text
  - Subtle use of gradients and glows for visual interest
  - Extremely fast load times — dark design paired with performance
  - Public design system as reference: https://vercel.com/design

## 5. Static Site Themes (Hugo & Astro)

### Hugo: PaperMod
- **URL**: https://github.com/adityatelange/hugo-PaperMod
- **Demo**: https://adityatelange.github.io/hugo-PaperMod/
- **Why notable**: The most popular Hugo theme. Clean, fast, well-maintained.
- **Key features**: Light/dark mode with system detection, built-in search (Fuse.js), cover images, auto TOC, breadcrumbs. Elegant simplicity — just a post list and reading view.

### Hugo: Blowfish
- **URL**: https://github.com/nunocoracao/blowfish
- **Demo**: https://blowfish.page/
- **Why notable**: Powerful, lightweight Hugo theme built with Tailwind CSS. Very actively maintained. Successor to Congo theme.
- **Key features**: Multiple color schemes, dark mode, thumbnails, nested menus, search, analytics integration. More feature-rich than PaperMod while staying clean.

### Hugo: Terminal
- **URL**: https://github.com/panr/hugo-theme-terminal
- **Demo**: https://themes.gohugo.io/themes/hugo-theme-terminal/
- **Why notable**: Retro terminal aesthetic. Dark by default. Strong personality.
- **Key features**: Monospace typography, terminal-style UI elements, dark color scheme. Good if you want a distinctive, hacker/developer vibe.

### Hugo: Risotto
- **URL**: https://github.com/joeroe/risotto
- **Demo**: https://risotto.joeroe.io/
- **Why notable**: Minimalist theme inspired by terminal ricing aesthetics. Ships with 16 color schemes (10 dark, 6 light) including Dracula, Gruvbox, Tokyo Night, Solarized.
- **Key features**: Zero JavaScript, fully responsive, extremely lightweight. The color scheme system is excellent for customization.

### Hugo: Blog Awesome
- **URL**: https://github.com/hugo-sid/hugo-blog-awesome
- **Why notable**: Fast, minimal blog theme with dark mode. Clean and modern without being flashy.
- **Key features**: Dark mode support, minimal design, good typography defaults.

### Astro: AstroPaper
- **URL**: https://github.com/satnaing/astro-paper
- **Demo**: https://astro-paper.pages.dev/
- **Why notable**: The most popular Astro blog theme. Minimal, accessible, SEO-friendly. Built-in light/dark mode with customizable color schemes.
- **Key features**: Fuzzy search, excellent SEO defaults, accessible markup, multiple color schemes configurable via CSS custom properties.

### Astro: Astro Ink
- **URL**: https://github.com/one-aalam/astro-ink
- **Why notable**: Inspired by Hugo's Ink theme. Aims to stay light while providing a beautiful minimal blog experience.
- **Key features**: Dark mode, 6+ color modes, clean typography, minimal footprint.

## 6. Key Design Principles to Apply

Based on this research, here are the patterns that the best designs share:

1. **Dark backgrounds should not be pure black.** Use #0a0a0a, #111, or #1a1a1a. Add subtle gradients or tonal shifts for depth. Linear and Vercel both do this well.

2. **Typography carries everything.** In minimal designs, font choice and sizing hierarchy IS the design. Sans-serif (Inter, Maison Neue, system fonts) for a modern feel; serif (Lora, Merriweather) for editorial warmth.

3. **Full-bleed photography.** For a running blog, the trail/mountain imagery is your biggest visual asset. Use it full-width as hero sections, not as small thumbnails.

4. **Whitespace (or "darkspace").** The best blogs let content breathe. Generous line-height (1.6-1.8), wide margins, and spacing between sections.

5. **Content-first homepage.** The trend is away from landing pages and toward surfacing posts directly on the homepage (van Schneider, Lee Robinson).

6. **Minimal navigation.** One horizontal bar or a simple hamburger. Categories can be tags, not a complex menu tree.

7. **No sidebar.** Almost every modern blog design has dropped the sidebar. Single-column reading view dominates.

8. **Subtle motion.** Page transitions (View Transitions API), hover effects, and micro-animations add polish without distraction.
